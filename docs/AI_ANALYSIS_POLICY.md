# AI Analysis Policy

Market Pulse AI analysis must be cautious, educational, and separated into facts, interpretation, risks, cautious action plan, and disclaimer.

## Required Fields

- trend
- bullish factors
- bearish factors
- risk score 1-10
- volatility level
- key support/resistance unavailable state when technical levels are not calculated
- what would invalidate the view
- cautious action plan
- disclaimer

## Required Tone

Allowed examples:

- น่าติดตาม
- รอจังหวะ
- เสี่ยงสูง
- เหมาะกับผู้รับความเสี่ยงได้
- ควรกำหนดแผนรับมือก่อนลงทุน

Avoid:

- guaranteed
- certain
- must buy
- sure profit
- direct guaranteed buy/sell predictions

## Output Structure

1. Facts
2. Interpretation
3. Risks
4. Cautious action plan
5. Disclaimer

Every analysis must include: `This is not financial advice.`
