# Alert Engine

Phase E prepares an alert evaluation engine without sending real notifications.

## Rule Types

- Price move
- Volatility spike
- News impact
- Economic event
- Risk score change
- Portfolio concentration
- AI recommendation change
- Earnings event

## Alert Output

Every generated alert includes:

- Why the user was alerted
- Evidence
- Confidence
- Severity
- Timestamp
- Affected assets

## Quiet Mode

Quiet mode supports:

- Only critical alerts
- Daily digest only
- Quiet hours
- Selected assets only

## Audit

The engine records:

- Rule triggered
- Notification generated
- Channel selected
- Delivery status
- Failure reason

No unnecessary personal data is stored.

## Limitations

Phase E does not connect live notification credentials and does not send outbound messages. Alert signals are educational monitoring signals, not financial advice.
