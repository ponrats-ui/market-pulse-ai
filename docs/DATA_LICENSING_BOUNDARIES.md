# Data Licensing Boundaries

## Rules

- Do not redistribute licensed exchange constituent files without permission.
- Do not store or commit provider credentials.
- Do not fabricate missing market data, news, calendar events, or fundamentals.
- Attribute provider source and timestamp where data is returned.
- Use only provider terms that permit the intended use.

## Current Boundary

The exchange master remains a partial supported universe. Full S&P 500, Nasdaq-100, and SET coverage requires verified and legally usable sources.
