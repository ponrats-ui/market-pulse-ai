# Freemium Model

Market Pulse AI keeps core market intelligence available for free. Premium is designed to sell convenience, monitoring, summaries, and saved time, not access to basic market information.

## Free Plan

- Market data
- Chart
- Watchlist
- Basic comparison
- Basic fundamentals
- Basic risk analysis
- Basic PIA analysis
- Paper portfolio

## Premium Plan

Premium is planned at 49 THB/month. Payments are not enabled in Phase E.

- Intelligent alerts
- Email alerts
- LINE alerts
- Telegram alerts
- Morning brief
- Evening wrap
- Weekly portfolio summary
- Portfolio monitoring
- Risk alerts
- Important news alerts
- Economic event alerts
- Saved alert rules
- Premium report export

## Product Principle

Premium features should help users monitor markets more efficiently. They must not create guaranteed predictions, personalized regulated advice, or hidden access to basic information that should remain free.
