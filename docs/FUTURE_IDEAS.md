## Idea Evaluation Criteria

Before an idea becomes roadmap scope, it should be evaluated against:

- Real user problem
- Trust impact
- Data availability
- Explainability
- Product fit
- Maintenance cost
- Security and privacy impact
- Production readiness risk

## Important Note

Future ideas should not distract from product quality. An idea becomes valuable only when it solves a real user problem and can be delivered without weakening trust.
