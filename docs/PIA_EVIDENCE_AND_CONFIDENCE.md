# PIA Evidence And Confidence

## Evidence

PIA uses provider-returned quote, history, fundamentals, risk, news, macro, and metadata where available. Missing data is listed explicitly.

## Confidence

Confidence depends on:

- quote availability
- historical data depth
- factor coverage
- score dispersion
- unavailable provider data

## Probability

The probability engine returns bullish, neutral, and bearish probabilities that total exactly 100%. These are evidence-weighted scenarios, not price predictions.
