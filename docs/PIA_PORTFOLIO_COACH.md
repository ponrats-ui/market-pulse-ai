# PIA Portfolio Coach

## Purpose

PIA Portfolio Coach gives educational observations about simulated holdings. It is not personalized regulated financial advice.

## Observations

The coach can flag:

- excessive concentration
- sector imbalance
- insufficient diversification
- high volatility
- high cash position
- currency exposure
- stale or unavailable quote data

Each observation includes evidence, severity, suggested action, limitations, and confidence.
