# PIA Reasoning Architecture

## Principles

PIA uses evidence before opinion, probability before prediction, adaptive weighting before static scoring, and transparent uncertainty before confident language.

## Flow

Evidence -> Feature Engineering -> Adaptive Weights -> Committee Opinions -> Probability Engine -> Chief Investment AI -> Investment Thesis -> Assistant Response

## Committee Roles

- Technical Analyst: trend, momentum, volatility, support/resistance, technical risks.
- Fundamental Analyst: quality, growth, profitability, cash flow, valuation.
- Macro Economist: interest rates, inflation, currency, liquidity, economic regime.
- News Analyst: company events, regulation, earnings, sector events, news impact.
- Risk Manager: downside, liquidity, concentration, event risk, uncertainty.

Each member returns facts, interpretation, opinion, confidence, evidence, missing data, and conditions that would change the opinion.
