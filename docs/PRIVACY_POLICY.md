# Privacy Policy

Effective date: July 13, 2026

Market Pulse AI is an educational financial research dashboard. This policy explains what information the application may process and how users should understand local storage, simulated portfolio data, market data providers, and embedded media.

## Information Collected

Market Pulse AI does not require an account for the current release candidate. The application does not intentionally collect names, email addresses, payment details, real brokerage credentials, or private financial account information.

The application may process:

- Asset symbols selected by the user
- Watchlist entries saved in the browser
- Recent searches saved in the browser
- Simulated portfolio entries saved in the browser
- Language preference saved in the browser
- Relax Mode preference saved in the browser
- Technical browser metadata automatically handled by hosting providers

## Local Storage

Market Pulse AI uses browser local storage for convenience. Stored values may include selected language, watchlist, recent searches, simulated portfolio holdings and transactions, and Relax Mode preferences.

Local storage remains on the user's device unless the browser, operating system, or user action synchronizes it elsewhere. Users can clear local storage through their browser settings.

## Portfolio Simulation

The portfolio feature is simulated paper trading only. Users should not enter real account credentials, bank information, brokerage passwords, or private personal financial information.

## Provider Usage

The application calls market data and information providers through the backend where available. Provider responses may include prices, historical data, fundamentals, news, macro data, calendar data, and unavailable states.

Provider availability, data timing, licensing, and completeness are controlled by the provider. Market Pulse AI does not guarantee real-time or complete data.

## YouTube Embeds

PIA Relax Mode can embed YouTube streams only after a user explicitly opens the panel and presses Play. YouTube embeds may cause the user's browser to communicate with YouTube or Google services according to their own privacy policies.

Market Pulse AI does not download, restream, or store YouTube audio.

## Cookies

The current application does not intentionally set application cookies. Hosting providers, YouTube embeds, browsers, or third-party services may use cookies or similar technologies under their own policies.

## Analytics

No custom analytics provider is enabled in the current release candidate.

## Third-Party Providers

Third-party services may include:

- Cloudflare Pages for frontend hosting
- Render for backend hosting
- Yahoo Finance or yfinance-derived market data
- Optional configured data providers such as Finnhub, Alpha Vantage, FRED, NewsAPI, Trading Economics, RSS feeds, and Alternative.me
- YouTube for optional Relax Mode embeds
- GitHub for source repository hosting

## Data Retention

Browser local storage remains until the user clears it or the browser removes it. Backend provider requests are not designed to store user portfolios or personal identities in the current release candidate.

## Contact

Project repository: `https://github.com/ponrats-ui/market-pulse-ai`

Please report privacy or security concerns through the repository's security process and avoid posting secrets or personal financial data publicly.
