# Roadmap After V1

## Immediate After Founder Approval

1. Merge feature/founder-production-final into main.
2. Push main to GitHub.
3. Deploy backend to Render.
4. Deploy frontend to Cloudflare Pages.
5. Run production smoke tests.
6. Create a release tag only after production verification passes.

## V1.1 Data Expansion

- Expand exchange master coverage with verified provider-ready symbols.
- Add an automated data quality check for unsupported or stale symbols.
- Improve exchange metadata for Thai, US, ETF, commodity, FX, and macro assets.

## V1.2 Provider Hardening

- Add provider health dashboards.
- Add retry and timeout observability.
- Add transparent provider status surfaces for news, calendar, and macro data.

## V1.3 Portfolio Reliability

- Add saved portfolio scenarios.
- Improve realized/unrealized calculation transparency.
- Add exportable audit views without broker integration.

## V1.4 Founder Operations

- Add release checklist automation.
- Add production smoke scripts.
- Add build artifact verification for Cloudflare and Render.

## Thai Summary

หลัง V1 ควรเน้นขยายข้อมูลจริง, เพิ่มความแข็งแรงของ provider, ทำ portfolio ให้ตรวจสอบได้มากขึ้น, และทำ release process ให้ปลอดภัยกว่าเดิม
