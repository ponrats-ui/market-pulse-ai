# Portfolio Theory

## Purpose

Portfolio theory studies how assets interact inside a portfolio.

The value of an asset depends not only on its individual qualities but also on how it affects diversification, risk, return, and exposure.

## Core Concepts

- Diversification
- Correlation
- Concentration
- Allocation
- Volatility
- Drawdown
- Risk-adjusted return
- Cash position
- Rebalancing

## Product Relevance

Market Pulse AI should eventually connect opportunity discovery to portfolio context.

An attractive asset may be less appropriate if it increases concentration, sector imbalance, currency exposure, or correlated risk.

## Limitations

Historical correlation can change. Portfolio metrics depend on data quality and time period.

Portfolio intelligence should be educational and cautious.
